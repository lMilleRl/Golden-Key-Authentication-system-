# Sequence Diagram: Взаимодействие в системе "Золотой ключик"

## Обзор

Эта диаграмма последовательности показывает пошаговое взаимодействие между актерами и объектами в системе "Золотой ключик".

## Актеры и участники

| Актер/Участник | Описание |
|-------------------|-------------|
| Буратино (B) | The main protagonist, puppet |
| Карабас-Барабас (KB) | The main antagonist  |
| Key (K) | A key for the hidden door |
| Door (D) | The hidden door |

## Interaction Steps

### Шаг 1: Поиск ключа
- Буратино находит ключ

### Шаг 2: Погоня
- Если Карабас догоняет, Буратино убегает:
  - **Карабас-Барабас** пытается догнать Буратино
  - **Карабас-Барабас** пытается перехватить ключ
  - **Буратино** убегает и скрывает ключ
  - Ключ не отдан
- Если Карабас догоняет, Буратино не убегает:
  - **Карабас-Барабас** догоняет Буратино
  - **Карабас-Барабас** перехватывает ключ
  - Ключ отдан

### Шаг 3: Поиск двери
- Буратино находит скрытую дверь
- Дверь становится обнаруженной

### Шаг 4: Открытие двери (финал)
- Буратино открывает дверь
- Ключ сверяется с отверстием замка (подходит ли ключ)
- Дверь отворяется

## Диаграмма

![Диаграмма последовательности](sequence.png)

```plantuml
@startuml
title Пошаговое взаимодействие в системе "Золотой ключик"

actor "Буратино" as B
actor "Карабас-Барабас" as KB
participant Key as K
participant Door as D

' 1. Нахождение ключа
B -> B : найтиОбъект("Key")
B --> B : Key

' 2. После того как Буратино нашёл ключ
alt Карабас догоняет, Буратино убегает
    KB -> KB : догнать(Буратино)
      KB -> KB : перехватитьКлюч(Буратино)
    B -> B : убежать()
    B -> B : скрытьКлюч()
    B --> KB : ключ не отдан
else Карабас догоняет, Буратино не убегает
    KB -> B : догнать(Буратино)
    KB -> B : перехватитьКлюч(Буратино)
    B --> KB : ключ отдан
end

' 3. Нахождение двери
B -> B : найтиОбъект("Door")
B --> B : Door
B -> D : обнаружить()

' 4. Открытие двери
B -> B : открытьДверь(Door)
  B -> D : открыть(Key)
    D -> K : compare(замок) : bool
    K --> D : true
  D --> B : true
@enduml
```
